自己做个站，因服务器运维、不同 blog 技术栈迁移等成本，在个人电脑搭完后就不了了之。后来的 github page 对于前端 er 来说少了运维工作，最近关注 rust 技术的时候了解到 astro 博客，由于对 rust 的好感瞬间看上了 astro，其配合 netlify 上线维护成本很低，个人认为这种类似 nextjs+云对个人及小型团队使用趋势很强，以后迁移应该也是小成本，于是就有了现在的线上个站博客。
