凭借着简单的状态管理、细粒度的响应性及良好性能，SolidJS正迅速成为web开发社区关注的焦点。Solid已成为其他JavaScript框架的标杆。
Solid 很好的满足了 React 开发者的使用诉求，就让我用这篇文章带你搞点事情，我们做个任务待办的小Demo。

### Prerequisites
在跟进这篇教程之前您需要先具备一些 Javascript、TypeScript, Node.js 模块(modules) 以及前端框架中的组件，等相关的知识。

## Why Solid?
如果你使用过 React,你会发现 Solid 很容易上手。在 React 宣发 Hooks 的时候我简直不要太愉快，我意识到它将能很好的[解决状态管理危机](https://blog.logrocket.com/react-hooks-state-management/)。Hooks 使的在组件内管理状态更容易，但是在全局状态管理依旧是复杂的。
